Learn to Node
=============

## Install

To install [node.js](https://nodejs.org/en/):

  * Website (All): [https://nodejs.org/en/download/](https://nodejs.org/en/download/)
  * Homebrew (Mac): `brew install node`

To install [mocha](https://mochajs.org/) for testing:

  * `npm install -g mocha`
